# Cloza プライバシーポリシー

最終更新日: 2026-02-11

本プライバシーポリシーは、CLOZA（以下「本サービス」）におけるユーザー情報の取扱いを定めるものです。
本サービスをご利用いただくことで、本ポリシーに同意したものとみなします。

## 2. 取得する情報

本サービスでは、以下の情報を取得する場合があります。

### 2.1 ユーザーが入力・投稿する情報

- アカウント情報（メールアドレス、ログイン識別子）
- プロフィール情報（表示名、プロフィール画像、性別、生年月日、身長、自己紹介等）
- クローゼット情報（衣類画像、カテゴリ、色、季節、ブランド、タグ等）
- バーチャル試着用情報（立ち姿画像、試着結果画像、関連メタデータ）
- コーディネート関連情報（依頼、提案、レビュー、評価、コメント）
- マーケット関連情報（出品情報、購入情報、配送先情報、取引メッセージ）

### 2.2 サービス利用に伴い自動取得する情報

- 端末情報（OS、端末種別、アプリバージョン、言語設定等）
- ログ情報（アクセス日時、操作履歴、障害ログ、IPアドレス等）
- 広告関連識別子（IDFA、AD_ID 等。許可された場合に限る）

### 2.3 外部認証・決済サービス経由で取得する情報

- Apple / Google ログイン時に連携されるアカウント情報
- Stripe / Stripe Connect を通じた決済・入出金に関するステータス情報
- RevenueCat を通じたサブスクリプション状態（有効期限、ステータス等）

※ クレジットカード番号などの機密な決済情報は、当社サーバーで直接保存せず、決済事業者側で管理されます。

## 3. 利用目的

取得した情報は、以下の目的で利用します。

- 本サービスの提供、本人確認、アカウント管理
- AIによるコーディネート提案、画像処理、バーチャル試着機能の提供
- 出品・購入・配送・精算等のマーケット取引処理
- チャット、レビュー、コミュニティ機能の運用
- 不正利用防止、セキュリティ確保、障害対応
- 利用状況の分析、品質改善、新機能開発
- お問い合わせ対応、重要なお知らせの通知
- 法令遵守および紛争対応

## 4. 外部サービスの利用

本サービスでは、機能提供のために以下の外部サービスを利用しています。

- Supabase（認証、データベース、ストレージ、リアルタイム通信、Edge Functions）
- Google Cloud / Vertex AI（AI提案・画像処理の一部）
- RunPod（バーチャル試着・画像生成処理の一部）
- Stripe / Stripe Connect（決済・送金・返金関連処理）
- RevenueCat（サブスクリプション状態管理）
- Apple / Google（外部認証）

これらの提供事業者は、各社の利用規約・プライバシーポリシーに基づいてデータを取り扱います。

## 5. 第三者提供

当社は、次の場合を除き、個人情報を第三者へ提供しません。

- ユーザー本人の同意がある場合
- 決済、配送、インフラ運用等の業務委託に必要な場合
- 法令に基づき開示が求められた場合
- 生命・身体・財産の保護のために必要で本人同意取得が困難な場合

なお、マーケット取引の遂行に必要な範囲で、取引相手に配送情報等が共有される場合があります。

## 6. 国外移転

本サービスで利用するクラウドサービスのサーバー所在地は日本国外を含む場合があります。これに伴い、取得情報が国外で処理・保管されることがあります。

## 7. 保存期間

- 情報は、利用目的の達成に必要な期間、または法令で定められた期間保存します。
- アカウント削除後は、法令上保存が必要な情報を除き、合理的な期間内に削除または匿名化します。
- バックアップ等の都合により、完全削除まで一定期間を要する場合があります。

## 8. 安全管理措置

当社は、情報漏えい、滅失、改ざん、不正アクセス等を防止するため、以下を含む合理的な安全管理措置を講じます。

- 通信の暗号化
- アクセス権限の制御
- ログ監査および障害監視
- 委託先管理

## 9. ユーザーの権利

ユーザーは、法令に基づき、自己情報の開示・訂正・削除・利用停止等を求めることができます。ご希望の場合は、アプリ内サポートまたは下記お問い合わせ窓口までご連絡ください。

## 10. 未成年者の利用

本サービスは13歳未満の児童を対象としていません。13歳未満のユーザー情報を取得したことが判明した場合、合理的な手続で速やかに対応します。

本サービスは、広告配信および効果測定のため、広告識別子やトラッキング許可情報を利用する場合があります。端末設定またはアプリ内設定で制御できる範囲については、ユーザー自身で変更可能です。

## 12. ポリシーの変更

本ポリシーは、法令改正やサービス内容の変更に応じて改定することがあります。重要な変更がある場合は、アプリ内通知または本ページで周知します。
